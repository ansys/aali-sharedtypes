============
Value Types
============

.. grid:: 1
   :gutter: 2

   .. grid-item-card:: Logical and Graph Values
      :class-card: sd-shadow-sm sd-rounded-md
      :text-align: left

      SharedTypes includes typed value structs used in GraphDB and internal processing:

      - `TypedValue`
      - `LogicalType`
      - `GraphValue`

      These types support dynamic input/output mapping and value tracking in workflows.
